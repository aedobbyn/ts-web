About

The Open Trade Statistics is an independent project that values reproducible research and provides tidy trade data. At the moment we focus on commodities data.

Our added value is that, unlike many other good trade data projects concerned with data visualization, we focus on data processing. Our goal is to make the analysis of this data easy.

We advocate for openness and collaboration, so our contribution is focused on:

Releasing all of our source code on GitHub
Releasing snapshots of the software we use
Sharing curated data (direct download and API)

Data

Our data source is UN Comtrade, a United Nations database, that collects data as reported by customs agencies.

Comtrade provides data organized under two great categorizations: The Standard International Trade Classification (SITC) and the Harmonized System (HS). Each of these has different revisions. For example the HS introduced newer codes since its release in 1992, in the years 1996, 2002, 2007 and 2012 (the latter not a part of the project at the present time).

If you need to analyze exports growth for the period 1980-2000 at commodity level, you'll have to convert the codes from SITC rev 2 from/to HS 1992/96/+ or vice versa due to data availability. (SITC codes were in use in the 80s but HS 1992 has only been used since 1992.)

Here we provide consistent and complete data that we already converted to HS 2007. We used imports and exports data to correct each other (i.e replace "false" missing or zero values), and we looked for countries that did not report under one classification but used an older one. In all cases we have done our best to maintain the finest level of granularity in the data classifications.
