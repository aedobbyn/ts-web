All the details about the API

You should probably start by inspecting api.tradestatistics.io/countries

The tables we provide at the moment are:

  • countries
  • products
  • reporters
  • rankings
  • yc (Year - Commodity)
  • yr (Year - Reporter)
  • yrc (Year - Reporter - Commodity)
  • yrp (Year - Reporter - Partner)
  • yrpc (Year - Reporter - Partner - Commodity)

countries returns the details associated to a country code such as country full name and continent.

products returns the details associated to a product code such as product full name and product group.

reporters by default returns the iso code of all reporting countries in 2016, use reporters?y=2010 or any value between 1962 and 2016 for a different year.

rankings works in the same way as reporters.

y* tables return information about countries and products, and they can be subsetted by year, reporter or partner. These tables requiere specific parameters such as yrpc?y=2016&r=chl&p=arg provided these queries have no default values.
